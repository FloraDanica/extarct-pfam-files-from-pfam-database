# extarct-pfam-files-from-pfam-database
-----------------------------------extarct pfam files from pfam database-------------------------------------------
requirement: 
1) linux, python3
2) pfam datase (ftp://ftp.ebi.ac.uk/pub/databases/Pfam/releases/Pfam32.0/)
3) pfam ID list, like 
"
PF18246.1
PF18358.1
PF18359.1
...
PF18404.1
"
run: python extract.pfam.py
